# 店注文システム



## ソフト名



店注文システム



## 概要



CLI（ターミナル）上で動作する店注文システム。

商品注文、注文確認、注文一覧表示を行うことができる。



## 機能



1.商品一覧表示

2. 商品の注文

3. 注文内容確認

4. 注文履歴表示

5. 合計金額表示



## 使い方

1. 番号１を入力して商品の一覧を表示させる
2. 番号2を入力して商品を選択して、個数を指定する
3. 番号3を入力して注文の履歴を確認できる
4. 番号4を入力して合計の金額を確認できる
5. 追加の注文がなければ番号5を入力して終了する
