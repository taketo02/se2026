\# design.md



\## 画面3つの説明

1.検索画面

キーワードを入力してデータを検索する



2\.入力画面

ユーザーが必要な項目を入力する画面。入力後、保存ボタンを押すとデータが登録される



3\.詳細画面

選択した店の詳細情報や予約数、コメントを表示する





\## データ3つ

1\.店ID

2\.予約情報

3\.コメント内容



\##処理の流れ

入力　→　確認　→　保存　→　表示



\# システム構成

画面　→　API　→　サーバ　→　DB
